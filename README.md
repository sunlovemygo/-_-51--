# 普中51单片机 - 蜂鸣器音乐播放器
## 项目介绍
基于普中科技STC89C52实验板，实现**蜂鸣器播放双歌曲**功能，支持按键切换，LCD1602实时显示歌曲名。

播放歌曲：
1. Bad Apple!!
2. 天空之城

## 硬件信息
- 主控：STC89C52RC
- 蜂鸣器：P2^5
- 按键1：P3^1 切换 Bad Apple
- 按键2：P3^0 切换 天空之城
- LCD1602：标准接口
- 晶振：11.0592MHz

## 功能说明
1. 开机默认播放 Bad Apple
2. 按键K1/K2可随时切换歌曲
3. 歌曲播放完毕自动循环
4. LCD1602显示当前播放歌曲名

## 代码结构
- main.c：主逻辑、歌曲列表、按键扫描、播放控制
- Timer0：定时器中断产生音调
- LCD1602：屏幕显示
- Delay：毫秒延时

## 原理
通过定时器0输出不同频率方波驱动无源蜂鸣器，根据乐谱数组的音调与节拍播放音乐。
